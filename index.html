lovekang，东南大学（SEU）硕士在读（研二），主攻 夜间图像去雾 与 底层计算机视觉。

我的研究方向聚焦于夜间有雾场景下的视觉恢复——低光照、复杂人工光源与大气散射共同作用，形成白天去雾方法无法处理的独特退化模式。我探索如何用深度学习穿透黑暗，还原清晰世界。

日常主力语言是 Python，PyTorch 是首选框架，搭配 NumPy、OpenCV，偶尔用 C++ 优化性能。我热衷于将学术研究落地到自动驾驶、安防监控等真实场景。同时正在探索如何将 Agent 技术与我的视觉工作结合，做一些有意义的事。

研究方向

夜间图像去雾

专注于夜间有雾场景的视觉恢复——低光照、非均匀光照与大气散射共同形成的复杂退化，挑战传统去雾方法的极限。

🌃

夜间去雾

应对夜间雾霾的独特挑战：低信噪比、人工光源导致的颜色失真、空间变化雾密度。开发联合去雾与光照增强的深度模型。

Python PyTorch
💡

光照感知复原

建模非均匀夜间光照（路灯、车灯、霓虹灯辉光）与雾散射的交互。设计将光照与透射率分离的先验。

Illumination Prior Physical Model
🌐

真实场景部署

弥合夜间去雾中合成数据与真实场景的域差距。探索自监督与半监督学习范式，在安防与自动驾驶场景中泛化。

Domain Adaptation Deployment
未来方向

视觉 × Agent

正在探索如何将 Agent 技术与底层视觉工作结合，构建能主动感知、推理和行动的智能视觉系统。

Agent-Enhanced Vision System

结合大语言模型的多模态能力与图像复原专业知识，构建能够理解图像退化、自主选择复原策略、并在复杂环境中进行推理和决策的 Agent 系统。目标是将传统"被动修复"升级为"主动感知-推理-行动"的智能视觉闭环。

LLM Agent Multi-modal Vision + Reasoning
